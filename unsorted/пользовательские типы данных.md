Пользовательский тип данных привязывается к схеме
Пользователь, создавший тип, становится его владельцем по умолчанию.
У типа данных обязательно должно быть уникальное имя.
Какие типы можно создать:
- составной тип
- перечисления
- диапазонный
- базовый
## Составной тип
Список атрибутов с типами данных
Пример 
```
CREATE TYPE address_type AS (street TEXT, city TEXT, country TEXT)
INSERT INTO customers (name, address) VALUES ('Вася', ('Пушкина','Питер','Раша'))
```
## Перечисление
```
CREATE TYPE name AS ENUM (значения)
```
## Диапазонные типы
Например диапазоны timestamp могут использоваться для представления временных интервалов (уже задан в postgres)
встроенные в postgres диапазонные типы:
- int4range
- int8range
- numrange
- int4multirange
- int8multirange
- nummultirange
- tsrange
- tstzrange
- daterange
## Базовый тип

Для его создания необходимы права суперпользователя
