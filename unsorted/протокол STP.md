# Принцип работы
## Выбор корневого моста (коммутатора)
Опирается на показатель Bridge Priority
По умолчанию 32768
Значения в диапазоне 0-61440 с шагом 4096
Чем меньше, тем выше приоритет
К этому значению добавляется Extended System ID (для определения приоритета и VLAN BPDU)
