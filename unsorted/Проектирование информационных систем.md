# Информационная система
Информационная система - коммуникационная система по сбору, передаче и обработке информации об объекте для реализации функции управления

Информационная система отслеживает бизнес процесс
Информационная система должна предоставлять пользователю удобный интерфейс
Информационная система должна отражать бизнес логику

Информационную систему хорошо описывать общим языком диаграмм

Информационная система состоит из:
- аппаратного обеспечения
- программного обеспечения
- персонала

Этапы создания информационной системы:
- as is (что есть)
- to be (что нужно)
## Диаграммы
### Семейство IDEF
#### IDEF0
1) функциональный блок
2) интерфейсная дуга
3) декомпозиция
4) глоссарий
#### IDEF3
1) единицы работ
2) связи
3) перекрёстки
4) объекты ссылок
