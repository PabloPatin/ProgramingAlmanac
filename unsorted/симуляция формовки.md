# Процесс штамповки металла 
Листовая штамповка – один из видов холодной обработки давлением, при котором листовой материал деформируется в холодном или подогретом состоянии.

Листовой штамповкой изготавливаются разнообразные плоские и пространственные детали – от мелких, массой от долей грамма и размерами в доли миллиметра (секундная стрелка часов), до средних (металлическая посуда, крышки, кронштейны) и крупных (облицовочные детали автомобилей).

Толщина заготовки при листовой штамповке обычно не более 10 мм, но иногда может превышать 20 мм, в этом случае штамповка осуществляется с предварительным подогревом до ковочных температур.

При листовой штамповке используют: низкоуглеродистые стали, пластичные легированные стали, цветные металлы и сплавы на их основе.

Листовая штамповка производится по следующим этапам

- **Подготовка материала:** Лист нержавеющей стали очищается от загрязнений и смазывается специальной смазкой. Смазка снижает трение между материалом и штампом, предотвращает заедание и образование задиров, улучшает качество поверхности и увеличивает срок службы штампа. Выбор смазки зависит от типа штамповки, марки стали и требуемого качества поверхности.
- **Установка штампа на пресс:** Штамп устанавливается на пресс и тщательно выравнивается.
- **Регулировка пресса:** Регулируется ход ползуна пресса, усилие прессования, скорость движения ползуна и другие параметры.
- Пуансон, приводимый в движение ползуном пресса, воздействует на лист стали, заставляя его деформироваться и приобретать форму матрицы. В зависимости от типа штамповки, может выполняться вырубка, пробивка, гибка, вытяжка или их комбинация.
# Задание материалов
Для того, чтобы в Abaqus задать свойства материала, необходимо переключиться на вкладку Property b и выбрать инструмент Create Material
![[Pasted image 20250407130852.png]]
Вот какие свойства необходимо задать для моделирования стали **AISI 304**:
- **Общие свойства:**
    - **Плотность (Density):** Примерно 7.9e-9 тонн/мм³. Важно для расчета массы детали и тепловой инерции.
    - **Модуль упругости (Young’s Modulus):** Примерно 2.0e5 МПа. Определяет жесткость материала в упругой области.
    - **Коэффициент Пуассона (Poisson’s Ratio):** Около 0.3. Отношение поперечной деформации к продольной.
    - **Коэффициент теплового расширения (Coefficient of Thermal Expansion):** Примерно 17.3e-6 /°C. Учитывает расширение материала при нагревании.
    - **Удельная теплоемкость (Specific Heat):** Примерно 500 Дж/(кг·°C). Количество тепла, необходимое для нагрева 1 кг материала на 1°C.
    - **Теплопроводность (Thermal Conductivity):** Примерно 16.3 Вт/(м·°C). Определяет, насколько хорошо материал проводит тепло.
Для моделирования пуансона и матрицы можно использовать объекты типа discrete rigid, которые не поддаются деформации.

# Ассемблирование
При помощи инструмента Create Instance размещаем на рабочем поле все детали (лучше добавлять как dependent, чтобы было удобнее потом делать сетку). 
Далее с помощью инструмента **Translate Instance** смещаем заготовку на 500 мм по оси z, для этого выбираем деталь и прописываем начало и конец вектора перемещения (0, 0, 0 -> 0, 0, 500) и нажимаем done.
Пуансон проще разместить с помощью инструмента, **Translate To**. Для этого указываем поочерёдно плоскости контакта: сначала пуансона, потом заготовки.
Далее указываем вектор, вдоль которого будет происходить сдвиг (0, 0, 0 -> 0, 0, -1) а затем расстояние между ними (можно указать 0, чтобы не создавать дополнительной расчётной нагрузки при симуляции). 
На этом сборку деталей можно считать оконченной.
# Шаги
В данной симуляции будет 3 шага:
1) Установка деталей в начальные позиции
2) Формовка буквы из заготовки
3) Извлечение заготовки из пресса и отображение результата
Первый шаг (initial) задаётся по умолчанию и его трогать нет необходимости. После него следует создать ещё два шага. Для этого надо зайти в модуль **Step** и с помощью инструмента **Step Manager** или **Create Step** создать два шага, длинной в секунду каждый. Для примера назовём их *Press* и *Show*.

# Нагрузка
Модуль Load предназначен для описания всех внешних воздействий на модель. С его помощью необходимо зафиксировать на месте матрицу и края заготовок, а так же задать перемещение пресса.
Представим, что края заготовки прижаты зажимом, который не позволяет им перемещаться.
Для того, чтобы закрепить объекты, или их части необходимо задать краевые условия. Выбираем инструмент **Create Boundary Condition** в качестве шага устанавливаем initial, выбираем категорию механических краевых условий, тип  Symmetry/Antisymmetry/Encastre и нажимаем continue. Далее следует выбрать боковые грани заготовки, которые следует закрепить. В появившемся окне выбираем ENCASTRE (U1=U2=U3=UR1=UR2=UR3=0), что означает зафиксировать по всем трём осям перемещения и по всем трём осям вращения.

# Сетка
Для того, чтобы провести симуляцию, необходимо задать каждой детали сетку. Именно опираясь на сетку компьютер будет просчитывать перемещение кадого элемента, так что это очень важный шаг.
Для каждой части следует настроить сетку отдельно. Для этого следует перейти в модуль **Mesh**.
## Сетка заготовки
Чтобы снизить вычислительную нагрузку следует разметить область заготовки, в которой ожидаются наибольшие искажения, для этого воспользуемся уже освоенный инструментарием Partition. Разметим прямоугольную область, внутри которой будет располагаться буква, а затем с помощью инструмента Partition Face: Project Edges спроецируем разметку на другую сторону.
![[Pasted image 20250408222120.png]]
Затем начнём создавать сетку. Для начала используем инструмент Seed Part и зададим приблизительный размер полигона в 15 мм.
![[Pasted image 20250408215937.png]]
Затем, следует выбрать инструмент Seed Edges, и выделить им все размеченные рёбра. В этой области сделаем сетку более частой, задав параметр Approximate element size = 5.
Затем перейдём к инструменту **Assign Mesh Control**, в нём выберем сетку типа tet (состоящую из тетраэдров).
После, создадим сетку при помощи инструмента **Mesh Part**.
## Сетка пуансона
Следующий этап - разметка пуансона.
Для него укажем огромный размер глобальной сетки в Mesh Part, чтобы минимизировать количество полигонов на плоскостях, а затем с помощью инструмента Mesh Region разметим сеткой с размером элементов 5 только те рёбра, которые непосредственно соприкасаются с заготовкой. 
В параметре Curvature control следует так же задать deviation factor = 0.05, это сделает сетку более частой в местах изгибов.
