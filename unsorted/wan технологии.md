| Термин          | Область применения                                                                                            | Особенности                                                                                                              |
| --------------- | ------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------ |
| **TDM**         | Открытая технология<br>Телефония, цифровые каналы связи, мультиплексирование нескольких потоков               | Делит общий канал на временные слоты; каждому потоку выделяется свой интервал времени                                    |
| **Frame Relay** | **Открытый протокол WAN**<br>Соединение филиалов и удалённых сетей через сеть оператора                       | Передаёт данные по виртуальным каналам; использует кадры переменной длины; быстрее X.25 за счёт меньшего контроля ошибок |
| **ATM**         | **Открытая технология передачи данных**<br>Операторские и магистральные сети, передача голоса, видео и данных | Использует ячейки фиксированного размера 53 байта; поддерживает QoS и предсказуемую задержку                             |
| **ISDN**        | **Открытая технология цифровой связи**<br>Цифровая телефония, передача голоса, данных и сигнализации          | Работает по телефонной линии; позволяет передавать голос и данные в цифровом виде                                        |
| **DSL**         | **Открытая технология доступа**<br>Широкополосный доступ в интернет по медной телефонной линии                | Использует высокие частоты телефонной линии; интернет и телефон могут работать одновременно                              |
| **PPP**         | **Протокол**<br>Соединения точка-точка, dial-up, PPPoE, PPPoA, выделенные линии<br>                           | Поддерживает установку соединения, инкапсуляцию протоколов и аутентификацию                                              |
| **T1/E1**       | **Стандарт цифровых линий**<br>Выделенные линии для передачи голоса и данных                                  | Фиксированная скорость и стабильный канал; T1 — 1.544 Мбит/с, E1 — 2.048 Мбит/с                                          |
| **CHAP**        | **Открытый протокол аутентификации**<br>Проверка подлинности пользователя в PPP-соединениях                   | Использует схему challenge-response; пароль не передаётся открытым текстом                                               |
| **MPLS**        | **Открытая технология коммутации**<br>Сети операторов, корпоративные WAN, VPN, QoS                            | Пересылает пакеты по меткам, а не только по IP-адресу; упрощает управление трафиком                                      |
| **DWDM**        | **Открытая технология оптической передачи**<br>Магистральные оптоволоконные сети, каналы между дата-центрами  | Передаёт несколько каналов по одному волокну на разных длинах волн; очень высокая ёмкость                                |
