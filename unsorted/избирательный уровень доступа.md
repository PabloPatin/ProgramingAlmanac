Строится матрица субъект-объект
Субъект - пользователь
Объект - файл

|     | O1  | O2  | O3  |
| --- | --- | --- | --- |
| S1  | R   | RW  | R   |
| S2  | RW  | RWX | RWX |
| S3  | RW  | R   | R   |
Владелец может назначить права другому пользователю, но делегировать свои права на файл не может (другой пользователь не сможет выдать права на этот файл третьему)
