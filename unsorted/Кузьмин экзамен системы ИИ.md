# Нечёткая логика
Нечеткая модель - модель, построенная на нечёткой логике

Лингвистическая переменная - переменная, значения которой это слова естественного языка

Нечеткий логический вывод - получение заключения в виде нечёткого множества с использованием нечёткой логики

Фаззификация - преобразование чётких значений в степени уверенности нечёткого множества

Высота нечёткого множества - максимальное значение функции принадлежности

Нормальное нечёткое множество имеет высоту h=1

Ядро - множество точек, для которых функция принадлежности $\mu=1$

Логика Белнапа имеет 4 значения: T(истинно), F(ложь), N(ни то не другое), B(оба)
## Операции над нечёткими множествами
Объединение A и B = max{$\mu_A(X), \mu_B(X)$}

Пересечение A и B = min{$\mu_A(X), \mu_B(X)$}

A является подмножеством B ($A \subset B$), если $\mu_A(u) \leq \mu_B(u)$

# Экспертные системы
Экспертная система - это компьютерная программа, моделирующая консультацию эксперта

PROSPECTOR - поиск месторождений

DENDRAL - структура хим соединений

MYCIN, PUFF - диагностика бактериальных инфекций
Значения в MYCIN представлены тройками

XCON - определение конфигурации компьютера

Базу знаний разрабатывает инженер по знаниям
База знаний - ключевая составляющая

Нельзя разработать экспертную систему в отсутствии экспертов

Разработка системы возможна, если большинство экспертах сходятся во мнениях

Разработка системы оправдана, если решение принесёт значительный эффект и количество экспертов мало

Экспертные системы работают с СИМВОЛЬНЫМИ и ЛОГИЧЕСКИМИ операциями
Задачи для экспертных систем НЕ носят АЛГОРИТМИЧЕСКИЙ характер

Эксперт обучает систему, пользователь - консультируется у неё

## Этапы создания системы
- идентификация - определение задач, которые подлежат решению
- концептуализация - производится анализ области, выявляются понятия, взаимосвязи и методы решения задач
- формализация - определение способов предоставления знания
- реализация - создание системы, последний этап

## Дескрипционная логика
Tbox - набор утверждений общего вида (термины)
Abox - набор утверждений частного вида (факты)
(Abox + Tbox) - база знаний
