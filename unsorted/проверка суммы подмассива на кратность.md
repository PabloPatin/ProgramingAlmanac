Для того, чтобы проверить, используем [[префиксная сумма|префиксную сумму]]
Суть в том, что если сумма подмассива кратна числу, то остатки от деления у сумм `prefix[i]` и `prefix[j]` равны, если i и j - левая и правая границы подмассива.
```python
arr = [...]
prefix = 0
mods = {0}
for v in arr:
	prefix += v
	mod = prefix % k
	if mod not in mods:
		mods.add(mod)
	else:
		return True	

```
