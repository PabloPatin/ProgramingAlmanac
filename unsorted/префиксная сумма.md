Можно посчитать сумму для элемента массива так
```python
arr = [1, 4, 2, 5, 8, 1, 9]
summ = 0
prefix = []
for val in arr:
	summ += val
	prefix.appeend(summ)

# тогда получаем следующее
prefix = [0, 1, 5, 7, 12, 20, 21, 30]
# чтобы вычислить сумму элементов с i по j
prefix[j+1] - prefix[i]
```
