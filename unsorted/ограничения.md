Ограничение в postgres - объект, такой же, как и таблица
# CHECK
Синтаксис
```
CREATE TABLE name (col type, CONSTRAINT name CHECK (condition))
ALTER TABLE ADD CONSTRAINT name CHECK (condition)
```
