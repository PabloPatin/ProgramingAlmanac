| Термин          | Что это                                  | Открытый / проприетарный         | Применение                                                               | Особенности                                                                                                              |
| --------------- | ---------------------------------------- | -------------------------------- | ------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------ |
| **TDM**         | **Технология**                           | Открытая, стандартная            | Телефония, цифровые каналы связи, мультиплексирование нескольких потоков | Делит общий канал на временные слоты; каждому потоку выделяется свой интервал времени                                    |
| **Frame Relay** | **Технология / протокол WAN**            | Открытая, стандартизованная      | Соединение филиалов и удалённых сетей через сеть оператора               | Передаёт данные по виртуальным каналам; использует кадры переменной длины; быстрее X.25 за счёт меньшего контроля ошибок |
| **ATM**         | **Технология передачи данных**           | Открытая, стандартизованная      | Операторские и магистральные сети, передача голоса, видео и данных       | Использует ячейки фиксированного размера 53 байта; поддерживает QoS и предсказуемую задержку                             |
| **ISDN**        | **Технология цифровой связи**            | Открытая, международный стандарт | Цифровая телефония, передача голоса, данных и сигнализации               | Работает по телефонной линии; позволяет передавать голос и данные в цифровом виде                                        |
| **DSL**         | **Технология доступа**                   | Открытая                         | Широкополосный доступ в интернет по медной телефонной линии              | Использует высокие частоты телефонной линии; интернет и телефон могут работать одновременно                              |
| **PPP**         | **Протокол**                             | Открытый стандарт                | Соединения точка-точка, dial-up, PPPoE, PPPoA, выделенные линии          | Поддерживает установку соединения, инкапсуляцию протоколов и аутентификацию                                              |
| **T1/E1**       | **Технология / стандарт цифровых линий** | Открытая, стандартизованная      | Выделенные линии для передачи голоса и данных                            | Фиксированная скорость и стабильный канал; T1 — 1.544 Мбит/с, E1 — 2.048 Мбит/с                                          |
| **CHAP**        | **Протокол аутентификации**              | Открытый стандарт                | Проверка подлинности пользователя в PPP-соединениях                      | Использует схему challenge-response; пароль не передаётся открытым текстом                                               |
| **MPLS**        | **Технология коммутации**                | Открытая, стандартизованная      | Сети операторов, корпоративные WAN, VPN, QoS                             | Пересылает пакеты по меткам, а не только по IP-адресу; упрощает управление трафиком                                      |
| **DWDM**        | **Технология оптической передачи**       | Открытая, стандартизованная      | Магистральные оптоволоконные сети, каналы между дата-центрами            | Передаёт несколько каналов по одному волокну на разных длинах волн; очень высокая ёмкость                                |
