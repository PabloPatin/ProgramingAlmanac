# Хранимые процедуры
Предварительно скомпилированные SQL-коды
## Свойства
- повторное использование кода
- производительность - их выполнение на сервере происходит быстрее за счёт прекомпиляции
- безопасность - можно ограничить доступ к данным
- упрощение логики
## Синтаксис
### Создание процедуры
DELIMITER //
CREATE PROCEDURE Name_Proc()
BEGIN
...
END //
DELIMITER;
### Вызов процедуры
call Name_Proc()

## Входные параметры
Объявляются с помощью оператора IN при создании и в дальнейшем обязательно указываются при выводе

## Выходные параметры
Объявляются с помощью ключевого слова OUT. Для просмотра необходимо передать аргументами с символом @ при вызове, а потом просмотреть с помощью SELECT
	call Name_Proc(@out_param)
	SELECT @out_param
