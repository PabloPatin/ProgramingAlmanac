Особый временный объект SQL, с помощью которого можно итерироваться по результатам запроса.
Курсоры используются в процедурах и триггерах (хранимых конструкциях).
# Синтаксис
DECLARE имя_курсора CURSOR FOR запрос\
Создание ссылки

OPEN имя_курсора
Открытие курсора

## FETCH имя_курсора INTO ...
Работа курсора

CLOSE имя_курсора
Закрытие курсора
