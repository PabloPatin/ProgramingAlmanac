Декоратор для функции, который создаёт новый объект объект [[archive/языки_программирования/python 3/библиотеки/сlick/классы/Command|Command]] с функцией в качесnве [[callback|колбэка]]. Также привяжет к функции все опции ([[archive/языки_программирования/python 3/библиотеки/сlick/декораторы/option|option()]]) и параметры([[Parameter|parameter()]]), которыми она задекорирована.

# Параметры
## name
Имя команды по умолчанию - имя функции с заменой знаков подчёркивания на дефис. Имя может быть передано первым аргументом.

## cls
Класс команды, которую необходимо создать

## kwargs
Через [[keyword argument|kwargs]] можно передать остальные аргументы класса [[archive/языки_программирования/python 3/библиотеки/сlick/классы/Command|Command]]
