Команды могут быть добавлены в группу несколькими способами:
- Декоратором @group.command()
- методом group.add_command(command)
```python 3
@click.group()
def cli():
    pass

@click.command()
def initdb():
    click.echo('Initialized the database')

@cli.command()
def dropdb():
    click.echo('Dropped the database')

cli.add_command(initdb)


if __name__ == '__main__':
    cli()
```
