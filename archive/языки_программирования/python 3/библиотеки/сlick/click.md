# Описание

Click - библиотека для написания интерфейсов командной строки. Она поддерживает вложенность команд, упрощает добавление справки и ленивую подгрузку команд.^def

# Установка

`pip install click`

# Основные концепции

Любая функция становится командой Click, когда обёрнута декоратором @click.command()
