128 битный адрес
	2001:0D88:0000:0000:ACAD:0000:0000:0010:00FF

Состоит из 8 [[хекстет|хекстетов]] по 16 бит каждый

Первые 3 хекстета идентифицируют провайдера
4 хекстет - идентифицирует подсеть
остальные - адреса устройств

Нет широковещательной адресации

Маску рекомендуется брать /64, так как адресов достаточно и не стоит их экономить

Разрешается сокращать несколько последовательных нулевых хекстетов символом :: один раз в адресе

Минимальный размер пакета 1280 байт

Протокол имеет встроенное шифрование IPSec

При подключении копмпьютера маршрутизатор выдаёт адрес с помощью протокола [[SLAAC]]
# Типы адресов
## Юникаст
Однозначно идентифицирует один интерфейс устройства
## Мультикаст
Адреса выделяются из диапазона FF00::0/8
## Эникаст
Могут быть назначены нескольким интерфейсам, но пакет доходит только до ближайшего члена группы, а не всем участникам
## Link-Local
Предназначены для использования только в локальном канале
Автоматически конфигурируются на всех интерфейсах
Является аналогом локальных IPv4 адресов
FF80::1
## Loopback
Для пересылке пакета себе самому адрес ::1
